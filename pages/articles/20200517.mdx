### デバッグ記録

マークダウンで、`summary` タグを使ったら警告が出る問題。  
マークダウンで summary を使うと props として `close` という boolean が渡されるが、React が提供する Details 用の JSX コンポーネントには `close` という props はなく、代わりに `open` がある。  
それに対応するために、以下のように、`close` を True/False 逆にして `open` に渡してあげると warning 消えた。

<details close>
<summary>click me</summary>

```typescript
type DetailsProps = {
  children: ReactChildren;
  close: boolean;
}

const Details = ({ children, ...props } :DetailsProps) => {
  return (<details open={!props.close}>{children}</details>)
}
```

</details>
